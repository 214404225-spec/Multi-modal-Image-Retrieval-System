## 五、总结与建议

### 5.1 成本差异总结

| 维度 | 差异 |
|------|------|
| Ollama硬件升级成本 | +1,400~1,800元 |
| Ollama年电费差异 | +25~50元/年 |
| API单次查询成本差异 | +25%（约+0.057元/次） |
| API年成本差异（日均100次） | +20,520元/年 |

### 5.2 推荐策略

1. **短期/小规模使用**：继续使用现有方案（Qwen2.5-3B + Qwen2.5-VL-3B）通过Ollama本地部署
2. **中期/中等规模**：考虑升级到Qwen3系列，仍使用Ollama本地部署以控制成本
3. **长期/大规模使用**：如果日均查询量超过500次，建议评估API调用方案，利用云服务的弹性扩展能力

### 5.3 迁移建议

如果决定升级到Qwen3系列，需要修改的代码位置：
- `intent_module/module.py`：修改 `model_name` 参数
- `fine_grained_retrieval_module/constants.py`：更新 `QWEN_VL_OLLAMA_MODEL` 常量
- `fine_grained_retrieval_module/vl_models.py`：可能需要更新模型加载逻辑
